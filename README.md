  The project is based on vehicle sales data, which provides a comprehensive collection of information pertaining to the sales transactions of various vehicles. The dataset includes specific information about each vehicle, such as its make, model, trim, manufacturing year, and the condition of the vehicles as well as their odometer readings, enabling analysis of how these factors influence selling prices and sales volume.

  Car sales files were created using Python, and Vehicle sales files were created using Power BI. The map is an add-on to the car sales file.
